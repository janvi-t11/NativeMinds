**Better solution — use a fixed local IP:**

1. Go to your **WiFi Router settings** (usually `192.168.1.1` in browser)
2. Find **DHCP Reservation** or **Static IP**
3. Assign a fixed IP to your PC's MAC address
4. Your PC will always get the same IP

---

**Easiest solution for development — use `ngrok`:**

```powershell
ngrok http 8000
```

This gives you a fixed public URL like `https://abc123.ngrok.io` that never changes. Update `backendUrl` once and forget it.

Install ngrok:
```powershell
winget install ngrok
```
